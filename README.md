# IsoEngine 🎮

Isometric game engine — Unity-style editor UI, tilted map, character control, NPC, drag-and-drop level editor. Single `index.html`, zero dependencies.

## Live demo
`https://<your-username>.github.io/<repo-name>/`

## Features
- Unity-style dark editor UI (titlebar, toolbar, inspector panel, status bar)
- 16×16 isometric tilted map — 8 tile types with animated water & lava
- Edit mode: click/drag to paint tiles
- Drag & Drop mode: drag from palette onto map
- Play mode: WASD/arrows to move, camera follows player
- NPC wizard: walk close to trigger dialogue
- Inspector panel: live object properties, player state, NPC status
- Scroll to zoom, middle-drag to pan, R to reset camera

## Controls (Play mode)
| Input | Action |
|---|---|
| WASD / Arrows | Move |
| Shift | Sprint |
| Scroll | Zoom |
| Middle drag | Pan |
| R | Reset camera |
